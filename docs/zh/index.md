## 🎯 关于本项目

**AI4SocialScience** 是专为社会科学学者、研究人员和学生打造的综合资源中心。我们不仅汇集了 **AI 技能**（工具），还整理了相关的 **学术论文**、**数据集** 和 **其他资源**，全方位支持您的研究工作流。

## 🔬 研究工作流

所有资源均按照典型的社会科学研究流程进行分类：

```mermaid
graph LR
    A[一站式解决方案] --> B[文献综述]
    B --> C[研究设计]
    C --> D[数据收集]
    D --> E[数据分析]
    E --> F[写作发表]
    F --> G[成果展示]
    G --> H[教学传播]
```
