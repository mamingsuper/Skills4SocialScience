# Open Science Skills

## Overview

This document provides links and information about the [Open Science Skills](https://github.com/scdenney/open-science-skills) repository, an excellent external resource for social science researchers.

## About Open Science Skills

Open Science Skills is a comprehensive collection of practices, workflows, and tools that support reproducible and transparent research.

**Repository:** https://github.com/scdenney/open-science-skills

## Key Features

- **Reproducible Research Workflows:** Step-by-step guides for creating reproducible research
- **Data Management Protocols:** Best practices for organizing, storing, and sharing research data
- **Open Science Practices:** Guidelines for pre-registration, open access publishing, and transparent methodology
- **Version Control:** Using Git and GitHub for research project management
- **Documentation Standards:** Creating clear and comprehensive research documentation

## Recommended For

- **Research Design Phase:** Planning reproducible studies from the start
- **Data Management:** Organizing data collection and storage protocols
- **Collaboration:** Setting up team-based research workflows
- **Transparency:** Implementing open science principles in your research

## Integration with Skills4SocialScience

This external resource complements the skills in this repository by providing:
- Theoretical foundations for open science practices
- Practical templates and checklists
- Community standards and best practices
- Tools for research transparency and reproducibility

## How to Use

1. Visit the [Open Science Skills repository](https://github.com/scdenney/open-science-skills)
2. Review the documentation relevant to your research stage
3. Implement recommended practices in your workflow
4. Combine with the skills in this repository for enhanced research capabilities
