# Statistical Analysis

Essential statistical techniques for social science research.

## Contents

### Descriptive Statistics
- **Central Tendency**: Mean, median, mode
- **Dispersion**: Standard deviation, variance, range
- **Distribution**: Skewness, kurtosis, normality tests
- **Visualization**: Histograms, box plots, summary tables

### Inferential Statistics

#### Hypothesis Testing
- **t-tests**: Independent samples, paired samples, one-sample
- **ANOVA**: One-way, two-way, repeated measures
- **Chi-square**: Tests of independence, goodness of fit
- **Non-parametric**: Mann-Whitney U, Kruskal-Wallis, Wilcoxon

#### Regression Analysis
- **Linear Regression**: Simple, multiple, assumptions
- **Logistic Regression**: Binary, multinomial, ordinal
- **Advanced Models**: Hierarchical, multilevel, mixed effects

#### Correlation
- **Pearson's r**: Linear relationships
- **Spearman's rho**: Monotonic relationships
- **Partial Correlation**: Controlling for variables

### Advanced Techniques
- **Factor Analysis**: EFA, CFA, PCA
- **Structural Equation Modeling (SEM)**: Path analysis, mediation
- **Time Series Analysis**: ARIMA, forecasting
- **Survival Analysis**: Kaplan-Meier, Cox regression
- **Cluster Analysis**: K-means, hierarchical

### Statistical Power and Sample Size
- **Power Analysis**: A priori, post hoc
- **Effect Sizes**: Cohen's d, eta squared, r
- **Sample Size Calculation**: For different tests

## Key Concepts
1. **Statistical Significance**: p-values, confidence intervals
2. **Effect Size**: Practical vs. statistical significance
3. **Assumptions**: Testing and handling violations
4. **Multiple Comparisons**: Bonferroni, FDR
5. **Missing Data**: Deletion, imputation, modern methods

## Software and Resources
- **R**: Comprehensive statistical computing
- **Python**: statsmodels, scipy, scikit-learn
- **SPSS**: Point-and-click interface
- **Stata**: Econometrics and social science
- **SAS**: Enterprise analytics
- **JASP/JAMOVI**: Free, user-friendly alternatives

## Best Practices
1. Check assumptions before testing
2. Report effect sizes, not just p-values
3. Consider practical significance
4. Use appropriate corrections for multiple testing
5. Validate results with different methods
6. Document all analysis decisions
