# Literature Review and Reference Management

Skills for conducting systematic literature reviews and managing references.

## Contents

### Literature Search Strategies
- **Databases**: Google Scholar, JSTOR, PubMed, Web of Science, Scopus
- **Search Terms**: Keywords, Boolean operators, wildcards
- **Citation Chaining**: Forward and backward searching
- **Systematic Reviews**: PRISMA guidelines, screening protocols

### Critical Reading
- **Reading Strategies**: Skimming, detailed reading, note-taking
- **Critical Analysis**: Evaluating methodology, findings, limitations
- **Synthesis**: Identifying themes, gaps, contradictions

### Reference Management Tools

#### Zotero
- **Features**: Free, open-source, browser integration
- **Best For**: General research, collaboration
- **Tips**: Use tags, collections, and notes effectively

#### Mendeley
- **Features**: PDF annotation, social networking
- **Best For**: PDF management, discovering research
- **Integration**: Microsoft Word, LaTeX

#### EndNote
- **Features**: Comprehensive reference management
- **Best For**: Large libraries, institutional use
- **Advantages**: Powerful search, organization features

#### Others
- **RefWorks**: Web-based, institutional
- **Papers**: Mac-focused
- **Citavi**: Knowledge organization

### Citation Management
- **Import**: From databases, PDFs, websites
- **Organization**: Folders, tags, smart collections
- **Annotation**: Highlighting, notes, summaries
- **Citing**: Word processors, LaTeX, plain text

## Systematic Review Process
1. Define research question
2. Develop search strategy
3. Screen titles and abstracts
4. Full-text review
5. Data extraction
6. Quality assessment
7. Synthesis and reporting

## Best Practices
- Start early, update regularly
- Maintain consistent naming conventions
- Back up your library
- Use cloud sync for access anywhere
- Keep notes within your reference manager
