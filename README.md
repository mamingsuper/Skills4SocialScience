This repository is organized into six key areas covering essential skills for modern social science research:

### 1. [Data Analysis](./data-analysis)
Master the tools and techniques for analyzing social science data:
- **Python** for data science (pandas, numpy, scipy)
- **R** for statistical computing (tidyverse, ggplot2)
- **SPSS** for traditional statistical analysis
- **Excel** for basic data work
- Best practices for reproducible research

### 2. [Research Methods](./research-methods)
Learn various research methodologies:
- **Quantitative Methods**: Surveys, experiments, secondary data analysis
- **Qualitative Methods**: Interviews, focus groups, ethnography, case studies
- **Mixed Methods**: Integration strategies and design types
- Research ethics and design principles

### 3. [Statistical Analysis](./statistical-analysis)
Develop expertise in statistical techniques:
- Descriptive and inferential statistics
- Hypothesis testing (t-tests, ANOVA, chi-square)
- Regression analysis (linear, logistic, advanced models)
- Advanced techniques (factor analysis, SEM, time series)
- Power analysis and effect sizes

### 4. [Data Visualization](./data-visualization)
Create compelling visual representations:
- Principles of effective visualization
- Tools: Python (matplotlib, seaborn), R (ggplot2), Tableau
- Chart types and when to use them
- Design best practices and accessibility

### 5. [Literature Review and Reference Management](./literature-review)
Efficiently manage academic literature:
- Literature search strategies and databases
- Critical reading and synthesis techniques
- Reference management tools (Zotero, Mendeley, EndNote)
- Systematic review methodology

### 6. [Writing and Communication](./writing-communication)
Communicate your research effectively:
- Academic writing (papers, theses, reviews)
- Presentation skills (conferences, posters)
- Citation styles and avoiding plagiarism
- Visual communication and data storytelling
