## ğŸš€ Getting Started

### Prerequisites

To use these skills, you'll need:
- Claude Code CLI installed ([Installation Guide](https://docs.anthropic.com/claude-code))
- Basic familiarity with command-line tools
- Access to Claude AI API (for some skills)

### Installation

1. **Clone this repository:**
   ```bash
   git clone https://github.com/mamingsuper/Skills4SocialScience.git
   cd Skills4SocialScience
   ```

2. **Browse available skills:**
   ```bash
   find _skills -name "SKILL.md"
   ```

3. **Install a specific skill:**
   - Follow the installation instructions in each skill's directory
   - Most skills can be installed via Claude Code CLI

4. **Use the skill:**
   ```bash
   # Example: Using the humanizer skill
   /humanizer "Your text here"
   ```

### Quick Start Guide

New to AI-powered research tools? Start here:

1. **For Complete Paper Writing:** Try [Claude Scientific Writer](_skills/writing/claude-writer/SKILL.md) for an integrated academic writing solution
2. **For Research Design:** Explore [Open Science Skills](_skills/design/open-science-skills/SKILL.md) to learn reproducible research practices
3. **Contribute Skills:** Help build this repository by adding your own research tools and workflows
4. **Join the Community:** Share your experiences and learn from other social science researchers

## ğŸ¤ Contributing

We welcome contributions from the social science community! Here's how you can help:

### Adding New Skills

1. Fork this repository
2. Create a new branch (`git checkout -b feature/new-skill`)
3. Add your skill to the appropriate category folder
4. Include detailed documentation (see [template](docs/skill-template.md))
5. Submit a pull request

### Reporting Issues

Found a bug or have a suggestion? Please [open an issue](https://github.com/mamingsuper/Skills4SocialScience/issues).

### Skill Submission Guidelines

- **Documentation:** Include clear usage examples and prerequisites
- **Testing:** Ensure your skill works across different environments
- **Licensing:** All contributions must be compatible with the MIT license
- **Quality:** Follow academic integrity and research ethics standards
