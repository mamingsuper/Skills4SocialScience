‚ö° What Are "Skills"?

Skills are specialized instruction files (SKILL.md) that teach an AI agent how to perform a specific research task with expert-level precision. Think of them as reusable research procedures ‚Äî written once, used repeatedly across projects.

1
Browse the Skills library

Find a skill matching your research task (e.g., "Stata Panel Regression", "Literature Review Summary Matrix").

2
Download the SKILL.md file

Click "Download SKILL.md" on the skill page and place it in your AI's skills directory.

3
Invoke via slash command

Type /skill-name your task description in your agent session.

# Without a skill ‚Äî generic output: User: "Run a regression on my panel data" AI: *writes basic lm() call without controls or SEs* # With stata-accounting-research skill ‚Äî expert output: User: "/stata-panel Run fixed effects on firm_data.dta" AI: ‚úì Detected panel structure: firm-year ‚úì Using reghdfe with firm+year FE ‚úì Clustering SEs at firm level ‚úì Generating LaTeX-ready output table

‚ë• FAQ

‚ùì Frequently Asked Questions

No coding is required for many skills, particularly those focused on writing, literature review, and research design. Skills labeled "Beginner" are designed for researchers without programming backgrounds. Analysis skills (Stata, R, Python) benefit from basic familiarity with the tool but don't require expertise ‚Äî the AI handles the code.
AI4SS is completely free and open-source (MIT licensed). The skills themselves are free. However, using AI coding agents like Claude Code requires an Anthropic API key or subscription, and Cursor requires its own subscription. Many agents offer free tiers sufficient for basic research use.
Regular prompts are one-off instructions you type each time. Skills are structured instruction files (SKILL.md) that encode expert methodology, common pitfalls to avoid, and expected output formats. They persist across sessions, are version-controlled, and can be shared across a research team. Think of them as codified research procedures rather than ad-hoc queries.
Most skills work with any AI coding agent that supports system prompts or custom instructions (Cursor, Windsurf, Gemini CLI, Cline, etc.). Each skill page lists compatible tools under the metadata tags. Skills designed specifically for Claude's slash-command system may need minor adaptation for other tools.
Use the Submit page to fill in a structured form that generates a GitHub issue with pre-formatted YAML. A maintainer reviews and merges it. You can also submit directly via pull request on GitHub.
Never share identifiable participant data with AI systems without explicit consent and IRB approval. For sensitive datasets, use local/offline AI models (e.g., Ollama with local Llama models). AI4SS includes papers on AI research ethics ‚Äî see the Ethics & Policy category in Papers. Always disclose AI use in your methods section per journal guidelines.
