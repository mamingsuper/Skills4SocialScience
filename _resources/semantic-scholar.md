## Semantic Scholar

Semantic Scholar by Allen Institute for AI uses NLP and machine learning to analyze and surface key insights from over 200M academic papers. It offers both a web interface and a free API for programmatic access.

**Features:**
- TLDR auto-summaries for papers
- Citation context and intent classification
- Author profile pages with influence metrics
- Free REST API for building research tools
- Covers all scientific disciplines
