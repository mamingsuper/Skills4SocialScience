## Connected Papers

Connected Papers generates interactive graphs of related academic papers based on co-citation and bibliographic coupling. Start with any paper and discover clusters of similar work you may have missed.

**Features:**
- Visual graph of 50 most-related papers
- Node size = citation count, node color = year
- Prior works + derivative works views
- Powered by Semantic Scholar database
