# Literature Review Agent

This skill enables AI agents to conduct systematic literature reviews following social science best practices.

## Capabilities
- **Search & Discover**: Query multiple databases (Google Scholar, Semantic Scholar, arXiv) with structured search strings
- **Screen & Filter**: Apply inclusion/exclusion criteria to large paper sets
- **Extract & Summarize**: Pull key findings, methods, sample sizes, and theoretical frameworks
- **Synthesize**: Build thematic evidence matrices and identify research gaps
- **Export**: Format citations for Zotero, Mendeley, or BibTeX

## Usage
```
/literature-review Find seminal papers on AI governance in political science from 2020-2025
```

## Best Practices
- Always verify AI-extracted claims against original papers
- Use structured search protocols (PRISMA or similar)
- Document search strings and databases for reproducibility
