# Claude Scientific Writer

An end-to-end academic paper writing solution using Claude AI.

## Overview

Claude Scientific Writer is a comprehensive skill for academic paper writing, supporting multiple social science disciplines with structured IMRAD format.

## Features

- End-to-end academic paper writing
- Literature integration and citation management
- Structured academic paper formats (IMRAD)
- Multiple discipline support

## Installation

```bash
mkdir -p ~/.claude/skills/claude-writer
# Copy SKILL.md to:
~/.claude/skills/claude-writer/SKILL.md
```

## Usage

```
/claude-writer Draft a social science paper outline on AI governance.
```
