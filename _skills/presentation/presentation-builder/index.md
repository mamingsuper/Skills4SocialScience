# Presentation Builder

This skill helps researchers create professional academic presentations from their research data and manuscripts.

## Capabilities
- **Slide Generation**: Create structured slide decks from paper outlines or research summaries
- **Data Visualization**: Generate charts, tables, and infographics from analysis results
- **Speaker Notes**: Draft talking points and transition notes for each slide
- **Format Support**: Output to reveal.js, Marp, LaTeX Beamer, or PowerPoint
- **Conference Adaptation**: Adjust for different time slots (5-min lightning, 15-min panel, 45-min keynote)

## Usage
```
/presentation-builder Create a 15-minute conference presentation from my paper on digital governance
```

## Tips
- Focus on 1 key finding per slide
- Use visuals over bullet points wherever possible
- Include methodology slides for methods-oriented audiences
