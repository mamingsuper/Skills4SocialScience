# Using Claude Code with Stata - An Economist's Guide

A guide by Aniket Panjwani on how to use Claude Code with Stata for economic research.

## Source

## Description
This guide provides practical instructions on integrating Claude Code, an AI coding assistant, with Stata for conducting economic research. It covers workflow optimization, code generation, and debugging strategies.

## Use Cases
- Using AI assistants for Stata programming
- Automating repetitive tasks in economic research
- Code review and optimization
- Learning best practices for reproducible research
