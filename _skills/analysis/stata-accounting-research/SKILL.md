# STATA Accounting Research Skill

A comprehensive skill for accounting research using STATA, providing syntax patterns from 126 peer-reviewed Journal of Accounting Research (JAR) replication files (2017-2025).

## Source
- GitHub: https://github.com/jusi-aalto/stata-accounting-research

## Description
This skill provides STATA syntax patterns and code examples commonly used in top-tier accounting research, extracted from replication files of peer-reviewed publications in the Journal of Accounting Research.

## Use Cases
- Data cleaning and preparation in accounting research
- Econometric modeling for accounting studies
- Replication of published accounting research
- Statistical analysis with STATA
