# Data Visualization

Learn to create effective visualizations for social science data.

## Contents

### Principles of Data Visualization
- **Visual Perception**: How people read charts
- **Chart Types**: When to use bar charts, line graphs, scatter plots, etc.
- **Color Theory**: Accessibility, contrast, meaning
- **Design Principles**: Simplicity, clarity, accuracy

### Tools and Software

#### Python
- **matplotlib**: Basic plotting
- **seaborn**: Statistical visualizations
- **plotly**: Interactive plots
- **altair**: Declarative visualization

#### R
- **ggplot2**: Grammar of graphics
- **plotly**: Interactive visualizations
- **shiny**: Interactive dashboards

#### Other Tools
- **Tableau**: Business intelligence and dashboards
- **Power BI**: Microsoft's visualization platform
- **Excel**: Basic charts and graphs
- **D3.js**: Web-based custom visualizations

### Best Practices
1. Know your audience
2. Choose appropriate chart types
3. Avoid chart junk
4. Label clearly
5. Maintain data integrity
6. Consider accessibility (colorblind-friendly)

### Common Visualization Types
- **Distributions**: Histograms, density plots, box plots
- **Comparisons**: Bar charts, grouped bars, small multiples
- **Relationships**: Scatter plots, correlation matrices
- **Time Series**: Line graphs, area charts
- **Compositions**: Pie charts, stacked bars, treemaps
- **Geographic**: Maps, choropleth, cartograms
