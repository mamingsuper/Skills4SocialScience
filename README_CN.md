## 🚀 关于本项目

**AI4SocialScience** (原 Skills4SocialScience) 现已全新升级！我们致力于打造社会科学学者的综合资源中心，汇集：
1.  **AI 技能 (Skills)**: 自动化研究的工具和 Agent。
2.  **论文 (Papers)**: 社会科学与 AI 交叉领域的学术论文。
3.  **资源 (Resources)**: 数据集、提示词 (Prompts) 和教程。

## 📂 项目结构

本项目按研究工作流组织：

*   **[_skills/](./_skills/)**: AI 工具和技能代码。
*   **[_papers/](./_papers/)**: 精选学术论文。
*   **[_resources/](./_resources/)**: 数据集和指南。
