## 🎯 关于本项目

本仓库包含了专为社会科学学者、研究人员和学生设计的 AI 驱动技能与工具的综合集合。无论您是在进行文献综述、分析数据，还是准备发表论文，都能在这里找到按研究工作流阶段组织的宝贵资源。

**核心特性：**
- ✅ 按研究工作流阶段组织
- ✅ 针对社会科学方法论量身定制
- ✅ AI 驱动的工具和自动化
- ✅ 双语文档（中英文）
- ✅ 持续更新，社区贡献

---

## 🔬 研究工作流

我们的技能按照典型的社会科学研究工作流程组织：

```
🎯 一站式解决方案（跨阶段集成工具）
        ↓
文献综述 → 研究设计 → 数据收集 → 数据分析 → 写作发表 → 成果展示 → 教学传播
```

## 🚀 快速开始

### 前置要求

要使用这些技能，您需要：
- 安装 Claude Code CLI（[安装指南](https://docs.anthropic.com/claude-code)）
- 基本的命令行工具使用能力
- 访问 Claude AI API（某些技能需要）

### 安装步骤

1. **克隆此仓库：**
   ```bash
   git clone https://github.com/yourusername/Skills4SocialScience.git
   cd Skills4SocialScience
   ```

2. **浏览可用技能：**
   ```bash
   cd skills/
   ls
   ```

3. **安装特定技能：**
   - 按照每个技能目录中的安装说明操作
   - 大多数技能可通过 Claude Code CLI 安装

4. **使用技能：**
   ```bash
   # 示例：使用 humanizer 技能
   /humanizer "您的文本"
   ```

### 快速入门指南

初次使用 AI 驱动的研究工具？从这里开始：

1. **完整论文写作：** 尝试 [Claude Scientific Writer](skills/0-one-stop-solutions/CLAUDE_SCIENTIFIC_WRITER.md) 获得集成的学术写作解决方案
2. **研究设计：** 探索[开放科学技能](skills/2-research-design/OPEN_SCIENCE_SKILLS.md)学习可重复研究实践
3. **贡献技能：** 通过添加您自己的研究工具和工作流来帮助建设本仓库
4. **加入社区：** 分享您的经验，向其他社会科学研究者学习

## 🤝 贡献指南

我们欢迎社会科学社区的贡献！您可以通过以下方式帮助：

### 添加新技能

1. Fork 此仓库
2. 创建新分支（`git checkout -b feature/new-skill`）
3. 将您的技能添加到相应的分类文件夹
4. 包含详细文档（参见[模板](docs/skill-template.md)）
5. 提交 Pull Request

### 报告问题

发现 Bug 或有建议？请[提交 Issue](https://github.com/yourusername/Skills4SocialScience/issues)。

### 技能提交指南

- **文档：** 包含清晰的使用示例和前置要求
- **测试：** 确保您的技能在不同环境下可用
- **许可：** 所有贡献必须与 MIT 协议兼容
- **质量：** 遵循学术诚信和研究伦理标准
